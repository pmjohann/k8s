# Add clusters

```
addcluster atlanta:1,dallas:1,frankfurt:1,fremont:1,london:1,newark:1,singapore:1,tokyo:1,toronto:1
```

# Apply manifests

```
applycluster
```

# Remove clusters

```
delcluster atlanta
delcluster dallas
delcluster frankfurt
delcluster fremont
delcluster london
delcluster newark
delcluster singapore
delcluster tokyo
delcluster toronto
```
